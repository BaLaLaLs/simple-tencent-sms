### 安装
```
cargo add simple-tencent-sms
```
### 使用
secret 获取相关可查看腾讯云官网文档
```rust
    let tencent_sms = TencentSMS::new(
        "secret_id".into(),
        "secret_key".into(),
        "sms_app_id".into(),
    );
```
#### 发送短信
```rust
    let response_json = tencent_sms
        .send_sms(
            Region::Beijing,
            "sign_name".into(),
            vec!["+86xxxxxxx"],
            "template_id".into(),
            vec!["xxxx"],
        )
        .await?;
```
response 和官方文档返回json一致
